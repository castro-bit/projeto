## 📖 Sobre

Este portfólio foi criado para exibir meus projetos pessoais de forma interativa. Ele carrega informações a partir de um arquivo JSON e apresenta cada projeto com uma imagem, nome e tags, permitindo fácil navegação.

## ✨ Funcionalidades

- Visualização de projetos pessoais com imagens e informações.
- Navegação para cada projeto ao clicar na imagem ou nome.
- Exibição de tags relacionadas a cada projeto.

## 🛠️ Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- JSON

## 🚀 Como Rodar

1. Clone o repositório:
   ```bash
   git clone https://github.com/castro-bit/portfolio.git
   ```
2. Navegue até o diretório do projeto.
3. Abra o arquivo `index.html` em seu navegador.

## 📂 Estrutura do Projeto

```
/portfolio
|-- index.html               # Página principal
|-- js/                      # Arquivos JavaScript
|-- css/                     # Arquivos CSS
|-- projects.json            # Dados dos projetos
|-- README.md                # Documentação
```
